# 🚀 HabitFlow

A full-stack habit tracking application built with:

- ⚡ FastAPI (Backend)
- ⚛️ React + Vite + Tailwind (Frontend)
- 🍃 MongoDB (Database)
- 🔐 JWT Authentication

---

## ✨ Features

- User Signup / Login
- JWT Authentication
- Create & Manage Habits
- Mark habits complete / undo
- Daily streak tracking
- Protected routes
- User-specific data isolation

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- FastAPI
- MongoDB (PyMongo)
- JWT (python-jose)
- bcrypt for password hashing

---

```bash
cd backend
pip install -r requirements.txt
```
