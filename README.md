# 🚀 HabitFlow

A full-stack habit tracking application built with:

- ⚡ FastAPI (Backend)
- ⚛️ React + Vite + Tailwind (Frontend)
- 🍃 MongoDB (Database)
- 🔐 JWT Authentication

---

## Features

- User Signup/Login
- JWT Authentication
- Create/Delete Habits
- Mark habits complete/undo
- Streak tracking
- Protected routes

---
